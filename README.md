# Real-Time Stock Market Analytics Dashboard

A professional portfolio-ready stock analytics dashboard built with Python, Streamlit, Plotly, pandas, NumPy, and yfinance.

## Features

- Live stock data using yfinance
- Stock selection from sidebar
- Latest stock price and market metrics
- Historical price charts
- 50-day and 200-day moving averages
- Volume analysis
- Daily return calculation
- Volatility calculation
- Multi-stock comparison
- Automated insights such as bullish, bearish, high volatility, and stable trend

## Tech Stack

- Python
- pandas
- numpy
- yfinance
- streamlit
- plotly
